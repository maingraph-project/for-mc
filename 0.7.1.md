## 0.7.1

- 编辑器设置新增「未保存时保存按钮高亮」开关：可开启或关闭保存按钮在未保存时的绿色脉冲高亮提示（默认开启）。
- 修复了因 Architectury 重构导致的节点丢失问题，恢复了以下事件节点：
    - 蓝图被调用(`on_blueprint_called`)：作为被调用蓝图的入口事件，接收调用者传递的参数
    - 玩家离开(`on_player_leave`)：当玩家离开游戏时触发
    - 玩家移动(`on_player_move`)：当玩家移动时触发，输出精确坐标和移动速度
    - 左键点击方块(`on_left_click_block`)：当玩家左键点击方块时触发
    - 拾取物品(`on_item_pickup`)：当玩家拾取物品时触发（仅NeoForge版本支持）
- 为以下节点加入了节点描述（在编辑器中悬浮节点时的显示说明）：
    - 加法 (+)(`add_float`)：返回两个数字 A 与 B 相加的结果。
    - 减法 (-)(`sub_float`)：返回 A 减去 B 的差。
    - 乘法 (*)(`mul_float`)：返回 A 与 B 相乘的积。
    - 除法 (/)(`div_float`)：返回 A 除以 B 的商；当 B 为 0 时返回 0 以避免除零错误。
    - 取模 (%)(`mod_float`)：返回 A 除以 B 的余数；当 B 为 0 时返回 0。
    - 幂运算 (A^B)(`pow_float`)：返回 A 的 B 次方（A^B）。
    - 表达式计算(`eval_expr`)：计算字符串中的算术表达式，支持 +、-、*、/、%、^ 以及括号；出错时返回 0。
    - 绝对值(`abs_float`)：返回输入数字的绝对值（非负值）。
    - 最小值 (Min)(`min_float`)：返回 A 与 B 中较小的值。
    - 最大值 (Max)(`max_float`)：返回 A 与 B 中较大的值。
    - 区间限定 (Clamp)(`clamp_float`)：将数值限制在最小值与最大值之间（包含边界）。
    - 四舍五入(`round_float`)：将数字四舍五入为最接近的整数（以浮点数返回）。
    - 向下取整 (Floor)(`floor_float`)：返回不大于该值的最大整数。
    - 向上取整 (Ceil)(`ceil_float`)：返回不小于该值的最小整数。
    - 随机浮点数(`random_float`)：返回介于 min（含）到 max（不含）之间的随机浮点数。
    - 随机整数(`random_int`)：返回介于 min 到 max（均含）之间的随机整数。
    - 随机布尔值 (概率)(`random_bool`)：以指定概率（0–1）返回 true，否则返回 false。
    - 向量加法(`vector_add`)：返回两个向量 A 与 B 按各分量（x、y、z 分别）相加的结果。若任一输入不是向量则返回零向量（各分量均为 0 的向量）。
    - 向量减法(`vector_sub`)：返回向量 A 减去 B 的结果。若任一输入不是向量则返回零向量（各分量均为 0 的向量）。
    - 向量乘法 (标量)(`vector_mul`)：将向量 A 的每个分量乘以普通数值 B（标量），返回缩放后的向量。
    - 向量除法 (标量)(`vector_div`)：将向量 A 的每个分量除以普通数值 B（标量）。当 B 为 0 时返回零向量（各分量均为 0 的向量）以避免除零。
    - 向量归一化(`vector_normalize`)：将向量 A 归一化（缩放到长度正好为 1）为单位向量（长度为 1 的向量）。若为零向量（各分量均为 0）则返回零向量。
    - 向量长度(`vector_length`)：返回向量 A 的长度（即模/大小），以普通数值（标量）表示。
    - 向量点积(`vector_dot`)：返回向量 A 与 B 的点积（也称内积），以普通数值（标量）表示。
    - 向量叉积(`vector_cross`)：返回向量 A 与 B 的叉积（也称外积）。
    - 向量距离(`vector_distance`)：返回向量 A 与 B 之间的直线距离（欧几里得距离），以标量表示。
    - 播放声音(`play_sound`)：在指定坐标处播放一段声音，可调节音量与音高。
    - 获取时间(`get_time`)：返回当前世界的游戏内时间（整数，单位为 tick，一游戏日 = 24000 tick）。
    - 设置时间(`set_time`)：将当前世界的游戏内时间设置为指定值（整数，单位为 tick，一游戏日 = 24000 tick）。
    - 获取天气(`get_weather`)：返回当前天气，结果为 clear（晴朗）、rain（下雨）或 thunder（雷暴）之一。
    - 设置天气(`set_weather`)：将当前天气设为指定类型（clear 晴朗 / rain 下雨 / thunder 雷暴），并可设置持续时间（单位为 tick）。
    - 浮点数(`float`)：输入一个数字（可带小数）作为固定数值并输出，供其他节点使用。
    - 整数(`int`)：输入一个整数（不带小数）作为固定数值并输出，供其他节点使用。
    - 布尔值(`boolean`)：设置一个“是/否”（真/假）开关并输出，供其他节点使用。
    - 字符串(`string`)：输入一段文字（文本）作为固定内容并输出，供其他节点使用。
    - 获取变量(`get_variable`)：根据变量名（名称）读取当前蓝图中已保存的变量数值并输出。
    - 设置变量(`set_variable`)：把某个数值保存到指定名称的变量中（仅当前蓝图运行期间有效），并继续向下执行。
    - 获取全局变量(`get_global_variable`)：根据变量名读取全局变量数值并输出，该数值会永久保存在存档中。
    - 设置全局变量(`set_global_variable`)：把数值保存到全局变量（永久保存在存档中），并继续向下执行。
    - 获取实体变量(`get_entity_variable`)：读取绑定到某个生物/实体身上的数值并输出，该数值随该实体保存在存档中。
    - 设置实体变量(`set_entity_variable`)：把数值保存到指定实体身上（随该实体保存在存档中），并继续向下执行。
    - 正弦(`sin`)：返回输入角度的正弦值。输入应为弧度（角度需先用“转弧度”节点转换）。
    - 余弦(`cos`)：返回输入角度的余弦值。输入应为弧度。
    - 正切(`tan`)：返回输入角度的正切值（正弦除以余弦）。输入应为弧度。
    - 反正弦(`asin`)：返回反正弦（已知正弦值求角度），结果以弧度表示，范围在 -π/2 到 π/2 之间。
    - 反余弦(`acos`)：返回反余弦（已知余弦值求角度），结果以弧度表示，范围在 0 到 π 之间。
    - 反正切(`atan`)：返回反正切（已知正切值求角度），结果以弧度表示，范围在 -π/2 到 π/2 之间。
    - 反正切2(`atan2`)：根据坐标 (y, x) 返回反正切，结果以弧度表示，范围在 -π 到 π 之间，并能正确判断所在象限。
    - 转弧度(`to_radians`)：将角度从角度制（度）转换为弧度（弧度制），供三角函数使用。
    - 转角度(`to_degrees`)：将角度从弧度（弧度制）转换为角度制（度）。
    - 字符串拼接(`string_concat`)：将两段文字（字符串 A 与 B）首尾相接合并为一段文字并输出。
    - 多字符串组合(`string_combine`)：把多个输入端口的文本按从上到下的顺序合并为一段文字（可点击按钮添加更多输入端口）。
    - 字符串长度(`string_length`)：返回文字（字符串）中包含的字符个数，以整数表示。
    - 字符串包含(`string_contains`)：判断一段文字中是否包含另一段子文字（子串）。
    - 替换字符串(`string_replace`)：把文字中所有的某段内容（旧文本）替换为另一段内容（新文本），输出替换后的整段文字。
    - 截取子字符串(`string_substring`)：从文字中截取从第 start 到第 end 个字符（索引）之间的一段，输出该子文字。
    - 大小写转换/修剪(`string_case`)：转换文字的大小写或去除首尾空格：UPPER 转大写、LOWER 转小写、TRIM 去除两端空白。
    - 分割字符串(`string_split`)：以指定的分隔符（如逗号）把一段文字切分成多段，输出为文本列表（字符串列表）。
    - 射线检测(`raycast`)：从起点沿某个方向射出一条看不见的射线（直线），检测最先碰到的方块或生物。可设置最大距离、是否在方块或生物处停下，并输出命中类型（MISS 未命中 / BLOCK 方块 / ENTITY 生物）、命中点坐标、命中实体、命中方块坐标，以及沿途经过的点位列表。
    - 等于(`compare_eq`)：判断两个数字 A 与 B 是否相等。
    - 不等于(`compare_neq`)：判断两个数字 A 与 B 是否不相等。
    - 相等（任意类型）(`compare_any`)：判断两个值是否相等，提供三种比对方式：忽略类型（只比内容）、宽松类型（自动转换数字/文本后比较）、严格类型（类型和值都必须一致）。
    - 大于(`compare_gt`)：判断数字 A 是否大于数字 B。
    - 大于等于(`compare_gte`)：判断数字 A 是否大于或等于数字 B。
    - 小于(`compare_lt`)：判断数字 A 是否小于数字 B。
    - 小于等于(`compare_lte`)：判断数字 A 是否小于或等于数字 B。
    - 逻辑与(`logic_and`)：逻辑与（并且）：只有当 A 和 B 都成立时，结果才为真。
    - 逻辑或(`logic_or`)：逻辑或（或者）：只要 A 和 B 中有一个成立，结果就为真。
    - 逻辑非(`logic_not`)：逻辑非（取反）：把成立变为不成立，把不成立变为成立。
    - 逻辑异或(`logic_xor`)：逻辑异或（互斥）：A 和 B 中恰好一个成立时为真，两个相同时为假。
    - 获取列表项(`get_list_item`)：从列表中按索引（从 0 开始）取出对应位置的元素并输出。
    - 添加到列表(`list_add`)：把一个元素追加到列表末尾，输出添加后的新列表。
    - 从列表删除(`list_remove`)：按索引删除列表中对应位置的元素，输出删除后的新列表。
    - 获取列表长度(`list_length`)：返回列表中元素的个数（以整数表示）。
    - 列表包含(`list_contains`)：判断列表中是否含有某个元素。
    - 设置列表项(`list_set_item`)：把列表中指定位置的元素替换为新值；若索引正好是列表长度，则追加到末尾，输出修改后的列表。
    - 合并列表(`list_join`)：用指定的分隔符（默认逗号）把列表中的元素连接成一段文字并输出。
    - 从列表中随机抽取(`random_list_item`)：从列表中随机取出一个元素并输出。
    - 给予物品(`give_item`)：给指定玩家发放指定物品（可设置数量与 NBT），随后继续执行。
    - 清除物品(`clear_item`)：清除指定玩家背包中的指定物品，可设置清除的数量（数量为 0 时清除全部）。
    - 获取物品数量(`get_item_count`)：返回指定玩家背包中某种物品的总数量（以整数表示）。
    - 拥有物品(`has_item`)：判断指定玩家的背包中是否含有某种物品。
    - 获取物品信息(`get_item_info`)：根据物品 ID 查询该物品的属性（最大堆叠数、最大耐久、是否为食物、是否为方块类物品），并分别输出。
    - 获取物品栏物品(`get_inventory_item`)：按索引（从 0 开始）读取指定玩家背包中某个格子的物品，输出其物品 ID、数量与 NBT。
    - 通过 UUID 获取实体信息(`get_entity_info_byuuid`)：通过 UUID 在世界中查找对应的生物/实体，输出其 UUID、名称、类型、注册名、坐标、生命值、最大生命值，以及是否为存活实体、是否为玩家、是否在线和权限等级。
    - 通过实体获取实体信息(`get_entity_info_byentity`)：直接以实体作为输入，输出其 UUID、名称、类型、注册名、坐标、生命值、最大生命值，以及是否为存活实体、是否为玩家、是否在线和权限等级。
    - 通过 UUID 获取实体(`get_entity_by_uuid`)：通过 UUID 在世界中查找对应的实体并输出该实体对象（找不到时输出空）。
    - Self（获取当前挂载目标）(`self`)：输出当前触发蓝图所挂载的实体（及其实体 UUID），常用于在蓝图内部引用自身。
    - 获取范围内实体(`get_entities_in_range`)：以某坐标为中心、在指定半径范围内查找所有实体，可按类型（注册名）筛选，输出实体列表。
    - 获取实体朝向(`get_entity_look_direction`)：获取指定实体当前朝向（视线方向）的单位向量并输出。
    - 类型转换(`cast`)：将一个值转换为指定的目标类型（在节点上选择类型）并输出。
    - 浮点数转整数(`to_int`)：把一个数值（浮点数）转换为整数并输出。
    - XYZ 分解(`break_xyz`)：把一个 XYZ 坐标（向量）拆分成 x、y、z 三个数值并分别输出。
    - 创建 XYZ(`make_xyz`)：把三个数值（x、y、z）组合成一个 XYZ 坐标（向量）并输出。
    - 分支(`branch`)：根据条件是否成立，走「真」或「假」分支并继续对应方向的逻辑。
    - 多分支(`switch`)：根据输入的控制值匹配对应分支并跳转执行；不匹配时走默认分支。
    - 有限循环(`for_loop`)：从起始值到结束值循环执行循环体，每轮输出当前索引，可用「跳出循环」提前结束，全部完成后走「循环完成」分支。
    - 等待 (刻)(`wait_tick`)：暂停执行指定的游戏刻数后再继续（基于游戏刻计时）。
    - 等待 (秒)(`wait_s`)：暂停执行指定的秒数后再继续（按 1 秒 = 20 刻换算）。
    - 计时器开始(`timer_start`)：开始计时（按计时器 ID），若已在计时则不影响；随后继续执行。
    - 计时器停止(`timer_stop`)：停止计时（按计时器 ID）并累计已经过的时间；随后继续执行。
    - 计时器重置(`timer_reset`)：清零并重置指定计时器（按计时器 ID）；随后继续执行。
    - 计时器(`timer_get`)：读取指定计时器（按计时器 ID）已累计的刻数、秒数与是否正在计时，并分别输出。
    - 跳出循环(`break_loop`)：请求中断当前所在的循环，使其提前结束。
    - 调用本存档其他蓝图(`call_blueprint`)：调用本存档中的另一个蓝图并传入参数列表，执行结束后输出结果列表并继续。
    - 返回列表到调用蓝图(`return_to_caller`)：把结果列表返回给调用当前蓝图的蓝图（在被调用蓝图内使用，需与「当其他蓝图调用时」配合使用）。
    - 获取方块(`get_block`)：获取指定坐标处的方块 ID（如 minecraft:stone）。
    - 设置方块(`set_block`)：在指定坐标放置一个方块。
    - 破坏方块(`break_block`)：破坏指定坐标处的方块（可选择是否掉落物品）。
    - 填充区域(`fill_area`)：在两点构成的长方体区域内填充指定方块。
    - 打印到聊天栏(`print_chat`)：向触发玩家发送一条聊天消息（无触发玩家时广播给全服玩家）。
    - 打印到控制台(`print_log`)：将消息输出到控制台、日志文件与 /mgmc log 缓存。
    - 以玩家身份运行命令(`run_command_as_player`)：以指定玩家（或触发玩家）的身份执行一条命令。
    - 以服务器身份运行命令(`run_command_as_server`)：以服务器（控制台）身份执行一条命令。
    - 播放粒子效果(`play_effect`)：在指定坐标播放一次粒子特效。
    - 爆炸(`explosion`)：在指定坐标引发一次爆炸（力度由半径决定）。
    - 传送实体(`teleport_entity`)：将实体传送到指定坐标。
    - 添加药水效果(`add_potion_effect`)：为生物添加指定药水效果（含时长、等级与粒子显示）。
    - 杀死实体(`kill_entity`)：立即杀死实体。
    - 伤害实体(`damage_entity`)：对实体造成指定数值的伤害。
    - 治疗实体(`heal_entity`)：为生物恢复指定数值的生命值。
    - 设置实体着火(`set_entity_on_fire`)：让实体燃烧指定的秒数。
    - 施加推力(`apply_impulse`)：给实体设置一个瞬时速度（推力）。
    - 清除所有药水效果(`clear_potion_effects`)：清除实体身上的全部药水效果。
    - 显示客户端 Toast 提示(`client_show_toast`)：在客户端屏幕上显示一个 Toast 提示（含标题与内容）。
    - 打开网页(`client_open_url`)：在客户端默认浏览器中打开一个网页链接。
