## v0.3.2

- **新增**:
    - **持久化变量系统**: 引入了全新的变量持久化方案，数据将跨蓝图执行、跨服务器重启自动保存。
    - **全局持久化变量**: 
        - 新增 `获取全局变量` 节点。
        - 新增 `设置全局变量` 节点。
        - 数据统一存储在世界存档目录下的 `mgmc_variables.json` 中。
    - **实体持久化变量**:
        - 新增 `获取实体变量` 节点，支持通过实体对象或 UUID 绑定数据。
        - 新增 `设置实体变量` 节点，实现针对特定实体的持久化状态跟踪（如等级、属性等）。
        - 数据统一存储在世界存档目录下的 `mgmc_entity_variables.json` 中。
    - **蓝图编辑器增强**:
        - **未知节点处理**: 现在无法识别的节点将渲染为“未知节点”，并保留其原始端口和连线。

- **调整**:
    - **官网域名变更**: 将模组内所有官网链接及文档链接后缀由 `opens.ltd` 统一变更为 `nb6.ltd`。

- **修复**:
    - **蓝图失效问题**: 修复了在退出并重新进入世界后，原本的或新创建的蓝图会失效的 Bug。该问题是由于服务器停止时过度清理了全局事件节点定义导致的。本问题发现者：[MCMOD.CN 東方華燈宴 (11楼)](https://www.mcmod.cn/class/24266.html)、[BILIBILI.COM 郫克清YrxyLiss13](https://space.bilibili.com/497664932)
    - **复制、删除、重命名失效**: 修复了蓝图管理器右键菜单复制、删除、重命名失效的 Bug。该问题是由于错误发送了内部name导致的。本问题发现者：[BILIBILI.COM 郫克清YrxyLiss13](https://space.bilibili.com/497664932)

## v0.3.2 (English)

- **New**:
    - **Persistent Variable System**: Introduced a brand new variable persistence solution. Data will be automatically saved across blueprint executions and server restarts.
    - **Global Persistent Variables**: 
        - Added `Get Global Variable` node.
        - Added `Set Global Variable` node.
        - Data is stored in `mgmc_variables.json` within the world save directory.
    - **Entity Persistent Variables**:
        - Added `Get Entity Variable` node, supporting data binding via entity objects or UUIDs.
        - Added `Set Entity Variable` node, enabling persistent state tracking (e.g., levels, attributes) for specific entities.
        - Data is stored in `mgmc_entity_variables.json` within the world save directory.
    - **Blueprint Editor Enhancements**:
        - **Unknown Node Handling**: Unrecognized nodes are now rendered as "Unknown Node" with preserved ports and connections.

- **Adjustments**:
    - **Official Website Domain Change**: Updated all official website and documentation links within the mod from the `opens.ltd` suffix to `nb6.ltd`.

- **Fixes**:
    - **Blueprint Failure Issue**: Fixed a bug where original or newly created blueprints would fail after exiting and re-entering the world. This issue was caused by excessive cleanup of global event node definitions when the server stopped. Reported by: [MCMOD.CN 東方華燈宴 (11th floor)](https://www.mcmod.cn/class/24266.html), [BILIBILI.COM 郫克清YrxyLiss13](https://space.bilibili.com/497664932)
    - **Copy, Delete, Rename Failure**: Fixed a bug where copy, delete, and rename actions in the blueprint manager's context menu were failing. This issue was caused by incorrectly sending internal names. Reported by: [BILIBILI.COM 郫克清YrxyLiss13](https://space.bilibili.com/497664932)
