## 0.5.3

- 修复了 `/mgrun <蓝图名> <event>` 命令触发时，`on_mgrun` 节点名称输出不正确的问题（之前错误输出为 `mgmc:on_mgrun`，现已修正为输出正确的 `<event>` 参数内容）。
- “类型转换”节点：移除了原有的“转换为类型”手动选择接口，改为自动判断转换。该节点现在采用单输入单输出设计，输出端口类型为 Any，会自动尝试将输入（如字符串）转换为最匹配的数据类型（Boolean, Float, XYZ 等）。

## 0.5.3

- Fixed an issue where the `on_mgrun` node name was output incorrectly when the `/mgrun <blueprint_name> <event>` command was triggered (previously incorrectly output as `mgmc:on_mgrun`, now corrected to output the actual `<event>` parameter content).
- "Type Conversion" node: Removed the original "Convert to Type" manual selection interface in favor of automatic inference. The node now features a single-input, single-output design with an `Any` output port type, which automatically attempts to convert the input (e.g., a string) to the most suitable data type (Boolean, Float, XYZ, etc.).
