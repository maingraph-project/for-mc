## 0.7.0

### 新增功能

#### 秒表节点 (Stopwatch)

- 新增 `timer` 秒表节点，支持多种控制操作：
  - `START` (启动): 开始计时，重置累计时间为 0
  - `PAUSE` (暂停): 暂停计时，保存已用时间
  - `RESUME` (继续): 继续计时
  - `STOP` (停止): 停止计时
  - `RESET` (重置): 重置所有状态
- 输出端口：
  - `ELAPSED_TICKS`: 获取已用刻数（整数）
  - `ELAPSED_SECONDS`: 获取已用秒数（浮点数，刻数/20）
- 计时状态使用节点运行时数据存储，支持多个独立秒表实例

#### 服务器命令

- 新增 `/mgmc openui` 服务器命令：
  - 管理员可通过该命令远程打开指定玩家的蓝图选择界面
  - 通过网络数据包 `OpenUIPayload` 触发客户端打开 UI
  - 需要管理员权限（权限等级 >= 2）

### 改进

#### 节点悬停提示

- 重构节点悬停提示系统，支持所有节点类型：
  - 显示节点标题、分类、模组来源等元信息
  - 采用现代玻璃样式设计（Glassmorphism）
  - 改进视觉布局和配色方案
- 新增 `node_category.mgmc` 等分类本地化键

#### 标记节点描述

- 为标记节点添加描述字段支持：
  - `NodeDefinition` 新增 `description` 属性
  - `NodeHelper` 新增 `description()` 方法用于设置节点描述
  - 按住 `Alt` 键悬停标记节点时显示描述提示框
  - 包含节点用途说明和"长按\[w]思索"操作提示

### 修复

#### UI 渲染层级

- 修复屏幕空间 UI 元素被节点和区域遮挡的问题
- 为各类 UI 元素添加 Z 轴偏移：
  - 快速搜索和标记编辑：+150
  - 顶部工具栏：+100
  - 节点菜单和上下文菜单：+250
  - 通知弹窗：+300
  - 设置面板：+400
- 使用 `pushPose/popPose` 确保层级正确嵌套

### 本地化

- 新增中英文本地化条目：
  - `node.mgmc.timer.name`: 秒表
  - `node.mgmc.timer.port.start/pause/resume/stop/reset`: 启动/暂停/继续/停止/重置
  - `node.mgmc.timer.port.elapsed_ticks/elapsed_seconds`: 已用刻数/已用秒数

## English Version

### New Features

#### Stopwatch Node

- Added `timer` stopwatch node with multiple control operations:
  - `START`: Start timing, reset accumulated time to 0
  - `PAUSE`: Pause timing, save elapsed time
  - `RESUME`: Resume timing
  - `STOP`: Stop timing
  - `RESET`: Reset all states
- Output ports:
  - `ELAPSED_TICKS`: Get elapsed ticks (integer)
  - `ELAPSED_SECONDS`: Get elapsed seconds (float, ticks/20)
- Timing state stored using node runtime data, supporting multiple independent stopwatch instances

#### Server Commands

- Added `/mgmc openui` server command:
  - Administrators can remotely open a player's blueprint selection interface
  - Triggers client UI via `OpenUIPayload` network packet
  - Requires admin privileges (permission level >= 2)

### Improvements

#### Node Hover Tooltips

- Refactored node hover tooltip system, supporting all node types:
  - Display node title, category, mod source and other metadata
  - Adopted modern glassmorphism design style
  - Improved visual layout and color scheme
- Added localization keys such as `node_category.mgmc`

#### Marker Node Descriptions

- Added description field support for marker nodes:
  - Added `description` property to `NodeDefinition`
  - Added `description()` method to `NodeHelper` for setting node descriptions
  - Hold `Alt` key while hovering over marker nodes to show description tooltip
  - Includes node usage instructions and "Long press [w] to ponder" hint

### Fixes

#### UI Rendering Layer

- Fixed screen-space UI elements being obscured by nodes and regions
- Added Z-axis offsets for various UI elements:
  - Quick search and marker editing: +150
  - Top toolbar: +100
  - Node menu and context menu: +250
  - Notification popups: +300
  - Settings panel: +400
- Used `pushPose/popPose` to ensure correct layer nesting

### Localization

- Added Chinese and English localization entries:
  - `node.mgmc.timer.name`: Stopwatch
  - `node.mgmc.timer.port.start/pause/resume/stop/reset`: Start/Pause/Resume/Stop/Reset
  - `node.mgmc.timer.port.elapsed_ticks/elapsed_seconds`: Elapsed Ticks/Elapsed Seconds
