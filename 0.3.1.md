## v0.3.1

- **新增**:
    - **Error-to-Client 实时反馈**: 当服务端蓝图运行报错时，直接在客户端 UI 弹出带节点 ID 的错误提示，并自动定位、高亮显示问题节点，终结“盲编”时代。
    - **蓝图运行日志指令**: 新增 `/mgmc log` 命令，打印前100条日志
- **优化**:
    - **并发安全加固**: 使用原子化操作 (`compute`, `computeIfPresent`) 重构了 `BlueprintRouter.java` 的路由表更新逻辑。消除了多线程环境下对映射表进行“检查后再执行”导致的竞态风险，确保了在高并发网络请求下路由数据的一致性。
