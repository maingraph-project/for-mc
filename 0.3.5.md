# 0.3.5: Make W Great Again Update

- **New**:
    - Added "Hold [W] to open documentation" feature: Hold the W key while a node is selected to open its documentation.
    - Wrote documentation for each node and registered the URLs in the node's `web_url` property.
- **Changes**:
    - Renamed node `mgmc:set_entity_velocity` (Set Entity Velocity) to `mgmc:apply_impulse` (Apply Impulse), and changed the port ID from `velocity` to `impulse`.
- **Fixes**:
    - Fixed NullPointerException during search caused by NULL values.
    - Fixed an issue where seconds were incorrectly labeled as ticks for adding potion effects.
